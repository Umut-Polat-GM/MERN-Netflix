# MERN-Netflix

A Netflix clone built using the MERN stack (MongoDB, Express.js, React, Node.js).

## Features

- User authentication and authorization
- Browse and search for movies and TV shows
- Watch trailers and full content
- Add movies and TV shows to your watchlist
- Responsive design
- The Movie Database (TMDb) API integration

## Dependencies

- react
- express
- mongoose

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/Umut-Polat-GM/MERN-Netflix.git
    ```
2. Navigate to the project directory:
    ```bash
    cd MERN-Netflix
    ```
3. Install backend dependencies:
    ```bash
    npm install
    ```
4. Install frontend dependencies:
    ```bash
    cd frontend
    npm install
    ```

## Usage

1. Start the server:
    ```bash
    npm run dev
    ```
2. Start the frontend:
    ```bash
    cd frontend
    npm run dev
    ```
